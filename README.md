# Sistema Escolar

Projeto desenvolvido para a disciplina de Desenvolvimento de Sistemas.

## Objetivo

Sistema desktop desenvolvido em Java com Swing para gerenciamento básico de alunos, disciplinas e notas. O projeto permite cadastrar alunos, cadastrar disciplinas e lançar notas, calculando automaticamente a média dos estudantes.

O sistema possui uma tela de login para acesso do professor e uma interface gráfica dividida em módulos para facilitar a navegação entre as funcionalidades.

## Como rodar o projeto

### Pré-requisitos

* JDK instalado na máquina

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
cd SEU-REPOSITORIO
```

### 2. Compile o projeto

```bash
javac tarefa.java
```

### 3. Execute o programa

```bash
java tarefa
```

## Funcionalidades

### Login

* Autenticação de professor
* Validação de usuário e senha

### Alunos

* Cadastro de alunos
* Visualização dos alunos cadastrados

### Disciplinas

* Cadastro de disciplinas
* Visualização das disciplinas cadastradas

### Notas

* Lançamento de duas notas por aluno
* Cálculo automático da média
* Exibição dos resultados

## Etapas de desenvolvimento

### Interface gráfica

O sistema foi desenvolvido utilizando Java Swing, aplicando conceitos como:

* JFrame
* JPanel
* JLabel
* JTextField
* JTextArea
* JButton
* ActionListener

### Lógica de programação

Durante o desenvolvimento foram utilizados:

* Estruturas condicionais
* Manipulação de eventos
* Entrada e saída de dados
* Conversão de tipos
* Cálculo de médias
